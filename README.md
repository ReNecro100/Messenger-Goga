# Мессенджер Георгий

![Маскот мессенджера Георгий](MessengerGoga/static/media/icon.png)

Простой мессенджер, позволяющий пользователям обмениваться текстом и изображениями, используя систему аккаунтов, и создавая для этого чаты, каналы и чаты личных сообщений (ЛС).

## Основные функции

- Аккаунты
  - Создание собственного аккаунта
  - Возможность редактирования и удаления аккаунта
  - Просмотр информации о своём аккаунте
- Сообщения
  - Отправка сообщений (текста или изображения)
  - Сохранение сообщений
  - Удаление сообщений
- Чаты
  - Создание чатов, куда могу писать все пользователи, и каналов, куда может писать только создатель
  - Создание чата личных сообщений (ЛС)
  - Перемещение между чатами
  - Добавление других пользователей в чаты и каналы при условии наличия ЛС с ними
  - Просмотр информации о чате
  - Возможность редактирования и удаления чатов, ЛС и каналов
  - Возможность выхода из чата
- Интерфейс
  - Дизайн в стиле Windows XP
  - Георгий-сфера на фоне
  - Разделение функций на разные окна
  - Перемещение окон

## Установка и запуск

1. Клонировать репозиторий
  ```
  git clone https://github.com/ReNecro100/Messenger-Goga.git
  ```

### Локально

2. Создать виртуальное окружение
  ``` 
  cd projectdjango
  python -m venv venv
  venv\Scripts\activate 
  ```

3. Установить зависимости
  ```
  pip install -r requirements.txt
  ```

4. Применить миграции
  ```
  python manage.py makemigrations
  python manage.py migrate
  ```

5. В `MessengerGoga/settings.py` заменить `CHANNEL_LAYERS`
```python
#Старый код
CHANNEL_LAYERS = {
    "default": {
        "BACKEND": "channels_redis.core.RedisChannelLayer",
        "CONFIG": {
            "hosts": [os.environ.get("REDIS_URL", "redis://localhost:6379")],
        },
    },
}

#Новый код
CHANNEL_LAYERS = {
    "default": {
        "BACKEND": "channels.layers.InMemoryChannelLayer"
    }
}
```
6. Запустить сервер
  ```
  python manage.py runserver
  ```

7. Перейти по адресу https://127.0.0.1:8000

### С помощью Docker

2. Запустить Docker Desktop

3. Запустить контейнер
  ```
  docker compose -f "messengergoga/docker-compose.yml" up --build
  ```

4. Перейти по адресу https://127.0.0.1:8080

## Описание страниц

- `/reg` - регистрация аккаунта
- `/log` - вход в созданный аккаунт
- `/wsschat/<chat_id>` - чат
- `/newchat` - создание нового чата
- `/editchat/<chat_id>` - редактирование текущего чата
- `/edituser/<user_id>` - редактирование текущего аккаунта


## Руководство пользователя

Если пользователь не зарегестрирован в системе, его перебрасывает на страницу `/log`. Для перехода на страницу регистрации нужно нажать на кнопку `Запись` в верхнем правом углу, после чего он окажется на странице `/reg`. После регистрации система попросит пользователя зарегестрироваться, повторно введя данные созданного аккаунта.

При входе на главную страницу, пользователь окажется в изначальном канале `/wsschat/1`. Для создания нового чата, требуется нажать на кнопку `Создать новый чат`, на новой странице `/newchat` заполнить форму, после чего сохраниться, нажав соответствующую кнопку.

Пользователь, создавший чат или канал, а также все участники ЛС, имеют возможность редактировать его, нажав на кнопку `Редактировать чат`. Перейдя на страницу `/editchat/<chat_id>`, можно изменить его название и описание. После этого чат можно сохранить или удалить. Между чатами можно перемещать с помощью кнопок в окне Твои чаты.

Для добавления пользователя в чат или канал, нужно создать с ним ЛС. Для этого нужно нажать на имя желаемого участника в окне информации о чате в списке Участники, после чего участник появится в списке на добавление под названием Добавить. Нужно просто нажать на участника в этом списке.

В окне Сообщения пользователь может отправлять сообщения - текст или изображения (другие файловые форматы отправлять нельзя). Для удаления своих сообщений, можно нажать соответстующую кнопку рядом с сообщением.

Также у пользователя есть возможность редактировать информацию о своём профиле. Нажав на кнопку `Редактировать профиль`, после чего, перейдя на страницу `/edituser/<user_id>`, можно редактировать своё описание (не видно другим пользователям) и имя. После этого аккаунт можно сохранить, выйти из него или удалить.
